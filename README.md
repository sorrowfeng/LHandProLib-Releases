# LHandProLib-Releases

This repository is a release mirror for `sorrowfeng/LHandProLib`.

Purpose:
- mirror release notes from the private source repository
- mirror uploaded release assets for public distribution after review
- keep this repository minimal so the auto-generated `Source code` archives do not contain the private SDK source tree

Notes:
- release assets in this repository are synchronized by GitHub Actions
- source releases are read from the private `LHandProLib` repository
- this repository should not contain the private project source code

Files in this repository:
- `.github/workflows/sync-releases.yml`
- `.github/scripts/sync_releases.py`
